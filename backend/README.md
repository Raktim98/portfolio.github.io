# Portfolio Backend Setup

## Install Dependencies

```bash
cd backend
npm install
```

## Create .env file

Copy .env.example and create .env

## Run Backend

```bash
npm run dev
```

## Frontend Run

```bash
npm install
npm run dev
```

## API Endpoints

POST /api/contact
GET /api/messages

## View Database Messages

Use MongoDB Compass or open:

http://localhost:5000/api/messages
